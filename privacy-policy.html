Политика конфиденциальности

Приложение: Улиточный путь
Пакет приложения: com.raccoon.snailroad
Дата последнего обновления: 9 июня 2026

Эта Политика конфиденциальности описывает, как приложение «Улиточный путь» обрабатывает данные пользователей.

1. Какие данные использует приложение

Приложение не требует регистрации аккаунта и не запрашивает у пользователя имя, адрес электронной почты, номер телефона или иные данные для создания учетной записи.

Внутри приложения локально на устройстве могут сохраняться игровые данные, необходимые для работы игры, например:

  • рекорд игрока;
  • количество леденцов;
  • выбранный скин;
  • список открытых или купленных скинов;
  • настройки звука;
  • режим управления.

2. Где хранятся данные

Перечисленные выше игровые данные сохраняются локально на устройстве пользователя с помощью механизмов хранения Android, включая SharedPreferences.

Разработчик не использует собственный сервер для хранения игрового прогресса в рамках базового сценария работы приложения.

3. Реклама и сторонние SDK

Приложение использует рекламные технологии Yandex Ads SDK для показа рекламы. При работе рекламного SDK сторонние сервисы могут обрабатывать технические данные устройства и приложения, необходимые для показа, ограничения частоты, измерения эффективности рекламы, борьбы с мошенничеством и диагностики.

Такие данные могут включать, например, идентификаторы устройства или рекламные идентификаторы, сведения о взаимодействии с рекламой, IP-адрес, технические параметры устройства и приложения, а также другую техническую информацию, которую сторонний рекламный SDK обрабатывает в соответствии со своей собственной политикой конфиденциальности.

Подробнее о правилах обработки данных рекламными сервисами Яндекса можно узнать на официальных страницах Яндекса: https://yandex.com/legal/confidential/

4. Передача данных третьим лицам

Разработчик не продает персональные данные пользователей. При этом данные, обрабатываемые сторонними рекламными SDK, могут передаваться соответствующим поставщикам рекламных и аналитических сервисов в пределах, необходимых для работы таких сервисов.

5. Безопасность

Разработчик стремится использовать только необходимые для работы приложения данные и ограничивать обработку данных тем объемом, который нужен для игрового процесса и монетизации через рекламу.

6. Хранение и удаление данных

Локальные игровые данные хранятся на устройстве пользователя до удаления приложения, очистки данных приложения или сброса прогресса внутри самой игры, если такая функция доступна.

7. Дети

Если приложение распространяется для аудитории, в которую могут входить дети, разработчик стремится соблюдать применимые требования Google Play и правила рекламных платформ.

8. Изменения политики

Разработчик может обновлять настоящую Политику конфиденциальности. Актуальная версия должна публиковаться по тому же URL, который указан в карточке приложения.

9. Контакты

По вопросам, связанным с конфиденциальностью и обработкой данных, свяжитесь с разработчиком по адресу: your-email@example.com

Перед публикацией в Google Play замените этот адрес на свой реальный контактный email.
